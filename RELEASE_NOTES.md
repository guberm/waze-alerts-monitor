# Waze Alerts Monitor v1.0.0

Initial release with core features:
- Real-time Waze alert notifications
- Voice alerts with TTS
- GPS location tracking
- 5 alert types (Police, Hazard, Accident, Speed Camera, Red Light Camera)
- Customizable filtering and radius selection
- Dark mode support
- Persistent monitoring with notifications

## Installation

Download the APK from Releases and install on your Android device.

### v1.0.0 (Initial Release)
